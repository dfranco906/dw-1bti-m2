### MARKDOWN
> Es un lenguaje de marcarcado que facilita la
> aplicación de un formato a un texto empleando
> un serie de caracteres de una forma especial.

### COMANDOS DE BASH
*Los comandos mas comunes en bash son los siguientes:*
1. **cd:** change directory
2. **ls:** list
3. **pwd:** print working directory
4. **mkdir:** make directory
5. **rm:** remove
6. **mv:** move
